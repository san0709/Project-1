# Skillbridge — Learning Platform UI

A responsive React + Vite landing site styled to a teal theme. Includes Banner, Header, Courses, Partner carousel, Testimonials, Pricing plans, Footer/contact and other UI building blocks with Tailwind-friendly classes and fallback CSS.

## Quick features
- Modern teal theme with shared CSS variables (tokens)
- Responsive components (Header, Banner, Partner, Courses, Benefits, Testimonials, Pricing, Footer)
- Tailwind-first components (work without Tailwind via fallback CSS)
- Accessible markup (semantic elements, keyboard-friendly controls)
- Pricing UI: Free / Pro / Pro Plus with monthly/yearly toggle and modal signup
- Partner logos marquee (reduced-motion friendly)
- Smooth interactions and consistent button transitions

## Tech stack
- React (Vite)
- Tailwind CSS (progressive — utilities available when installed)
- PostCSS + Autoprefixer
- Plain CSS fallback for components
